# Much Ado about Prompting — Companion materials

Companion landing page for the paper *"Much Ado about Prompting: Text Classification in Experimental Economics"* (Çelebi & Penczynski).

## What's here

- **`index.html`** — landing page with three buttons.
- **`presentation.pdf`** — the slides.
- **`pipeline.html`** — interactive viewer of the codebook-to-prompt pipeline (Filter → Separate → Compress → Assemble) for all four tasks.
- **`viewer.html`** — interactive prompt-assembly viewer: pick a task, content level, format, framing, examples, etc., and see the assembled prompt.

Both HTML files are fully self-contained (data embedded). No build step or external assets.

## Hosting

Push to GitHub and enable GitHub Pages on `main` / root to serve at `https://<user>.github.io/prompting/`.
